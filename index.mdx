## What is Tokenami?

Tokenami is a modern CSS-in-JS library that replaces atomic utility classes with **atomic CSS variables**. It provides type-safe design tokens with autocomplete, zero runtime injection, and no bundler requirements—giving you the best of both CSS-in-JS and utility-first frameworks.

## Quick Example

```tsx
import { css } from '@tokenami/css';

function Button() {
  return (
    <button style={css({
      '--padding': 4,
      '--background': 'var(--color_primary)',
      '--hover_background': 'var(--color_primary-dark)',
    })}>
      Click me
    </button>
  );
}
```

## Why Tokenami?

<AccordionGroup>
  <Accordion title="No Runtime Injection">
    Following React team's recommendation to use static CSS files and inline styles for dynamic values. Tokenami generates CSS at build time with no runtime overhead.
  </Accordion>
  
  <Accordion title="No Class Soup">
    Your components have one class name for reusable styles, keeping HTML clean and DRY—unlike utility-first frameworks that repeat classes across components.
  </Accordion>
  
  <Accordion title="Type-Safe Tokens">
    Your theme defines and enforces design tokens with full TypeScript support and autocomplete—removing unused tokens shows errors in your IDE.
  </Accordion>
  
  <Accordion title="Portable & Extensible">
    Atomic CSS variables give consumers control to override styles. No specificity wars, no locked-in architecture.
  </Accordion>
</AccordionGroup>

## Key Features

- **Atomic CSS Variables** — Replace utility classes with CSS variables
- **Type-Safe Design Tokens** — Autocomplete and type checking for your theme
- **Zero Bundler Setup** — Simple CLI generates static styles
- **Composable Variants** — Build reusable component styles with variant support
- **Custom Selectors** — Define expressive selectors for hover, focus, and more
- **Grid-Based Spacing** — Consistent spacing system with grid values
- **Multiple Themes** — Built-in support for dark mode and theme switching
- **Framework Agnostic** — Works with React, Vue, Solid, Qwik, and more
